This lightweight and customizable plugin empower you to personalize user login experience and enhance visual appeal of your WordPress login page.

== Description ==

= Features =

- **Logo Upload:** Easily upload your custom logo to replace the default WordPress logo on the login page.
- **Background Color Picker:** Choose your preferred background color using an intuitive color picker.
- **Rounded Form Option:** Enable a modern look for your login form with a 1em border radius and no border width.

== Installation ==
**Installation**

1. Download the plugin ZIP file.
1. Upload and activate the plugin on your WordPress site.
1. Navigate to the Settings > Login Page Styler to customize your login page.
